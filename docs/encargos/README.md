# Encargos de la sesion de cotejo y conciliacion

Esta carpeta agrupa los encargos a Claude Code redactados durante la sesion de
trabajo sobre el motor de cotejo y la conciliacion diaria. Cada archivo sigue el
formato estandar: Contexto, CAMBIOs numerados, Tests, Commit y Verificacion manual.

## Indice

- `extraccion-dnis.md` — Extraer DNI/NIF de adquirente y transmitente en CTI y
  modelo 620; normalizar CET="NO" como ausente.
- `motor-cruce-declarativo.md` — Motor de cruce de datos por tabla de reglas
  (transferencia simple + herencia); identidad CRITICA, nombres ADVERTENCIA.
- `conciliacion-diaria.md` — Modulo de conciliacion: planilla Tempus, expedientes
  y hoja de caja por gestoria/dia.
- `telegram-encargo-2.md` — Agente de Telegram con respuestas detalladas y
  contexto de verificaciones.
